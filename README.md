# Сайт игр по медиабезопасности для образовательных организаций

Статический лендинг для презентации услуг по проведению игровых занятий и тренингов по медиабезопасности и медиаграмотности для школ, колледжей и вузов.

## Структура

- `index.html` — основная страница с описанием форматов, целевой аудитории, тем и процесса работы.
- `styles.css` — стили в современном тёмном/нейтральном оформлении.
- `script.js` — лёгкая логика для моковой формы заявки.

Форма заявки демонстрационная: данные никуда не отправляются, по нажатию «Отправить заявку (мок)» показывается сообщение об успешной отправке.

## Как запустить локально

1. Откройте папку проекта:
   - через Finder или любой файловый менеджер;
   - или в терминале:

   ```bash
   cd /Users/sanchez/site
   ```

2. Откройте файл `index.html` в браузере:
   - двойным кликом по файлу;
   - или перетащите файл в окно браузера.

Дополнительный сервер не требуется — это полностью статический сайт без бэкенда и без биллинга.
